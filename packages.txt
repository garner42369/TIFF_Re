# ==========================================================
# Streamlit Community Cloud 容器 (Debian Linux) 专有配置文件
# ==========================================================
# 
# ⚠️ 重要更新：虽然我们使用 Rasterio（自带预编译 GDAL），
# 但在某些 Streamlit Cloud 容器环境中，Rasterio 的 wheel
# 可能仍然需要系统级的 GDAL 头文件来完成安装或链接。
#
# 因此，我们需要添加以下系统依赖作为保底：
# ==========================================================

# GDAL 系统库和头文件（Rasterio 可能需要）
libgdal-dev
gdal-bin

# Python 开发头文件
python3-dev

# 基础编译工具（作为极端情况下的保底）
build-essential
